# 邮箱授权说明

Plutus 通过 IMAP 只读采集招商银行通知邮件。邮箱授权码不是邮箱登录密码，只用于第三方客户端访问邮件。

授权码必须在 Plutus 本地设置页填写：

```text
http://127.0.0.1:8973/#config
```

不要把邮箱授权码发给 Hermes、Agent 或聊天窗口。

## 支持的邮箱

### QQ 邮箱

在设置页选择 QQ 邮箱，填写 QQ 邮箱地址和 IMAP/SMTP 授权码。

官方入口：

```text
右上角头像 → 设置 → 账号与安全 → 安全设置 → 开启服务 → 生成授权码
```

## 163 邮箱

在设置页选择 163 邮箱，填写 163 邮箱地址和客户端授权码。

官方入口：

```text
设置 → POP3/SMTP/IMAP → 新增授权密码
```

也可以在网易邮箱大师中生成：

```text
我 → 邮箱管理 → 第三方登录管理 → 通用授权码 → 新增授权码
```

## Gmail

在设置页选择 Gmail，填写 Gmail 地址和 Google 应用专用密码。

官方入口：

```text
Google 账号 → 安全性 → 应用专用密码
```

## 本地保存位置

Plutus 会把授权信息保存在本机：

```text
~/.plutus/secrets/mail_auth.json
```

这个文件已被 `.gitignore` 排除，不会提交到仓库。
