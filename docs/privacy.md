# Plutus 隐私与数据处理说明

本文说明 Plutus 在当前实现中保存哪些数据、哪些数据可能离开本机，以及相关处理由谁控制。它描述的是实际技术边界，不代表 Hermes、AI 模型服务商、邮箱服务商或微信的隐私政策。

## 一句话说明

Plutus 的 SQLite 账本和邮箱授权码保存在用户自己的 Mac 上，Plutus 项目不提供保存用户账本的云端服务器。但“本地账本”不等于“所有数据始终不离开本机”：借记卡邮件解析、AI 分类、Hermes 查询和微信通知会按功能需要处理或发送部分交易数据。

## 本地保存的数据

默认安装目录是 `~/.plutus`，主要包括：

| 数据 | 默认位置 | 用途 |
|---|---|---|
| 交易、进项、分类、知识和同步状态 | `~/.plutus/plutus.db` | 本地账本与增量同步 |
| 邮箱类型、地址和授权码 | `~/.plutus/secrets/mail_auth.json` | 连接用户选择的邮箱 |
| 本地配置 | `~/.plutus/config.toml` | 服务、账本、统计和通知配置 |
| 运行日志 | `~/Library/Logs/plutus` | 本地故障排查 |

邮箱授权码文件和配置文件由安装流程设置为仅当前用户可读写。授权码不应发送给 Hermes、AI 模型、微信、聊天窗口或项目维护者。

## 邮箱访问

Plutus 使用用户提供的邮箱授权码，通过 IMAP 连接 QQ 邮箱、163 邮箱或 Gmail：

- 邮箱以只读模式打开，Plutus 不删除、移动或修改邮件。
- Plutus 搜索招商银行通知邮件，并在本地记录消息标识和 UID 水位以避免重复采集。
- 邮箱地址和授权码会在连接邮箱服务商时用于认证，这是完成 IMAP 访问所必需的。
- 邮箱服务商对登录、访问日志和邮件数据的处理受其自身政策约束。

## Hermes 和 AI 处理

Plutus 通过用户本机安装的 Hermes 执行以下任务：

### 借记卡邮件解析

信用卡「每日信用管家」邮件由 Plutus 的确定性解析器在本地解析。借记卡「一卡通账户变动通知」会将通知正文和接收时间交给 Hermes，用于识别：

- 支出、转账、进项或忽略类型；
- 卡尾号、金额、时间、交易对手、渠道、进项类型和备注。

通知正文可能包含姓名、卡尾号、金额、商户或付款方等财务信息。

### AI 分类

执行 AI 分类时，Plutus 会把以下内容交给 Hermes：

- 商户名称；
- 交易笔数和金额提示；
- 分类清单及说明；
- 用户写入知识库的分类规则和事实。

Plutus 只保存 AI 建议，用户可以确认或纠正。Hermes 是否调用外部 AI 模型、数据发送到哪里、是否保留以及保留多久，取决于用户自己的 Hermes 和模型服务商配置。安装前应检查相应配置及服务商政策。

### Hermes 查询和纠正

Hermes 通过本机 MCP 接口读取完成用户请求所需的账本结果，例如待分类交易、月度汇总和进项记录，也可以写入分类、备注和知识。返回结果可能包含交易时间、卡尾号、金额、商户、分类或进项来源。

## 微信通知

启用微信通知后，Plutus 会通过 Hermes 向用户配置的微信目标发送交易摘要，可能包含：

- 信用卡或借记卡类型及卡尾号；
- 交易金额；
- 商户名称；
- 分类或待确认状态；
- Plutus 内部交易编号。

用户在微信中查询或纠正账本时，消息和回答还会经过 Hermes 与微信。相关数据处理受 Hermes 配置和微信政策约束。不希望这些信息进入微信时，不应启用微信通知或通过微信查询账本。

## Plutus 项目不会主动做的事

当前版本：

- 不提供用于集中保存用户账本的 Plutus 云端服务器；
- 不包含广告或第三方用户行为分析 SDK；
- 不主动上传邮箱授权码；
- 不自动向项目维护者发送账本、日志或崩溃报告；
- 未经用户明确操作，不自动回填历史邮件。

## 本地 Web 控制台

Web 控制台默认监听 `127.0.0.1`，仅供本机访问。用户如果把监听地址改为 `0.0.0.0` 或以其他方式暴露端口，应同时设置访问口令，并自行承担局域网、代理和公网暴露带来的访问风险。

## 用户可控项

用户可以：

- 在邮箱服务商处撤销授权码，停止 Plutus 访问邮箱；
- 停止 Plutus 的 daemon 和 Web 服务；
- 在 Hermes 中移除 Plutus MCP 或关闭相关模型与微信通道；
- 备份后删除 `~/.plutus/plutus.db`、`~/.plutus/secrets/` 和本地日志。

删除本地文件不会自动删除已经发送给 Hermes 所配置模型服务商或微信的数据。如需删除这些数据，应按对应服务商提供的方式处理。

## 提交问题前

公开提交 Issue、截图或日志前，应删除邮箱地址、授权码、邮件正文、姓名、卡号、卡尾号、金额、商户、交易时间和微信目标等敏感信息。项目维护者不会要求用户提供邮箱授权码。

## 说明更新

产品的数据流发生变化时，本文也应同步更新。用户应在安装或升级前检查最新版本。
